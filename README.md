# ImpedanceAnalyzer

A minimal open source UI for the impedance analyzer by Wim Dijkman published in Elektor Magazine 2017 https://www.elektormagazine.com/magazine/elektor-201709/40741

This tool implements the communication protocol for the analyzer as defined and implemented by Win Dijkman (Thank you for making this open). The original version of his software doe not run on current windows versions, so I impemented a minimal version (initially measures only impedance over frequency with no GUI) for personal use. Might be extended in the future.
